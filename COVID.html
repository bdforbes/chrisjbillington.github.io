Exponential projections are based on most recent 5 days of data for each country
User of ICU bed numbers is a crude measure of when each country's healthcare system is under strained. Not all cases require an ICU bed, but neither are all ICU beds available (and may not be the limiting factor in treating critical cases), so this is very approximate.
Source for case numbers: Johns Hopkins COVID-19 data on github
